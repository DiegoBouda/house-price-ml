# House Price Prediction (Machine Learning)

## Overview
A machine learning application that predicts real estate prices using supervised learning models trained on housing data.

## Technologies
- Python
- scikit-learn
- pandas
- Flask

## Models
- Random Forest Regressor

## Model Interpretability
Feature importance analysis was performed using the Random Forest model to identify which input variables had the greatest impact on price predictions.

## Evaluation
Model performance evaluated using MAE and RMSE metrics.

## How to Run
pip install -r requirements.txt  
python main.py  
python app.py
